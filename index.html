Tabs - 6 in total

  • Gather Tab (1)
  • Collect Tab (2)
  • Planters Tab (3)
  • Status Tab (4)
  • Tools Tab (5)
  • Settings Tab (6)

Gather Tab (1)

  • Fields
    To edit a field, click the pen icon, to paste pre-made settings, click the 6 dots icon, and click “Paste”. Find pre-made settings in the Discord server → #patterns-configs
    – Editing the Field (Suggested to copy and paste pre-made settings from the Revolution Discord)
    – Please ensure "Max Backpack" is set to "95%". Sometimes the macro stays in the field too long when the backpack is 100%, causing the macro to freak and reset.
    Disable drift detection (uses supreme saturator to align char) → click the pen icon → click “align” under max backpack → disable

  • Boosts (Make sure sprinkler is in slot 1)

  • Settings
    – General
    Set sprinkler to your current sprinkler in Bee Swarm
    If you are blue and macro in Pinetree, enable “Use Fast Pine Return Path”

    Keep everything else the same
    – Conversion
    Customizable to your liking

    – Field Defaults
    You don’t have to touch this

  • AI Gather
    – Settings
    Enable “Use Full Model”, “Eco Mode” (Recommended). “Loot Mode” optional
    – Token Priority (customizable)

Collect Tab (2)

  • Dispensers – Enable “Wealth Clock”, “Glue Dispenser”, “Blueberry/Strawberry Dispenser”
  • Bug Run – If you have gifted vicious bee set respawn to 15%
  • Quests – Enable if lacking Polar Power
  • Buffs – Optional settings
  • Bosses – WIP
  • Blender – WIP

Planters Tab (3)

  • Duration 12–24 hours recommended
  • Clear Schedule resets all
  • Refresh Schedule refreshes all
  • Enable “Gather Loot”, “Reset when Harvesting”
  • Auto Planters → Enabled (Harvest every 3 hours)
  • Nectar Goals → choose per your macro time
  • Fields & Planters → Custom

Status Tab (4)

  • Logs – troubleshooting
  • Webhook – setup instructions

Tools Tab (5)

  • Auto Jelly
  • RBC Gather
  • Tad Sync
  • Vic Hop
  • Mondo Hop

Settings Tab (6)

  • Macro → Dark mode, hotkeys, capture mode, misc
  • Game → Movement, routing, join
  • Networking → Tad Sync
  • Presets → WIP
  • Accounts → WIP
